API Documentation

Welcome to Isabellas API. Below you will find details on how to interact with our endpoints.

Endpoints

Get Post

Endpoint: GET /products/id

Description: Retrieves a specific post by ID.

Example Request:

fetch('https://jsonplaceholder.typicode.com/posts/1')
    .then(response => response.json())
    .then(data => console.log(data));

Example Response:


    "Id": "Sample id (can contain both letters and numbers).
    

    "Brand": "Sample brand such as (Dior).
    

    "Item": "Samble item such as (Sneakers).
